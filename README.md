# Meme Quest

A responsive static website built using HTML and CSS.

## Installation Requirements

- A modern web browser (Chrome, Firefox, Edge, Safari)
- Optional: Visual Studio Code with the "Live Server" extension

## Running the Project Locally

1. Download or clone the project files.
2. Open the project folder in Visual Studio Code.
3. Right-click on `index.html`.
4. Select **"Open with Live Server"**  
   (or simply double-click `index.html` to open it in your browser).


## Online Version

The website is hosted and can be viewed here:

https://meme-quest-cyan.vercel.app


## Operating Instructions

- Use the navigation menu to move between pages.
- Scroll to view all content.
- Buttons and links are clickable for navigation.
- The layout is responsive for desktop and mobile screens.


## Limitations

- The website is static (HTML/CSS only).
- No backend or database functionality.
- No user authentication or form submission.
- Content is not dynamically generated.
